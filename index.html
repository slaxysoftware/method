API Nedir?

📦 API Çekme & Telegram Bot Eğitim Notları

1. API nedir?

API aslında bir köprü. Sen soru soruyosun o da sana cevap veriyo. Mesela "hava nasıl" diyosun → sana "32 derece" diyip dönüyo.

2. GET - POST olayı

GET → Veriyi al (mesela hava durumu)

POST → Veri gönder (mesela form doldur)

Kafanda kalsın: GET = al, POST = gönder

3. Python'da API çekme

import requests

url = "https://jsonplaceholder.typicode.com/posts/1"

r = requests.get(url)

print(r.json())

4. Telegram bot açma

1. Telegram'da @BotFather'a gir

2. /newbot yaz, bot adı ver

3. Sana bir TOKEN veriyo, bu şifre gibi

5. Botla API bağlama

import telebot

import requests

TOKEN = "BURAYA_TOKENİNİ_YAZ"

bot = telebot.TeleBot(TOKEN)

@bot.message_handler(commands=['start'])

def start(message):

    bot.reply_to(message, "Selam! /hava yaz hava durumunu söyleyim")

@bot.message_handler(commands=['hava'])

def hava(message):

    url = "https://api.open-meteo.com/v1/forecast?latitude=41.0&longitude=29.0¤t_weather=true"

    r = requests.get(url).json()

    derece = r["current_weather"]["temperature"]

    bot.reply_to(message, f"Şu an sıcaklık {derece}°C")

bot.polling()

6. Mantık

Kullanıcı /hava yazıyo → Bot API'ye soruyo → API cevap dönüyo → Bot cevabı kullanıcıya yolluyo

7. Pratik fikirler

/usd → dolar kuru

/film → rastgele film öner

/hava → sıcaklık

API = bilgi kaynağı, Bot = aracı, Sen = işi çözen adam
